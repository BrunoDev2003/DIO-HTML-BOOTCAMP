Agrupamento de seletores

Pódemos aplicar as mesmas regras CSS para vários seletores diferentes. Para isso, podemos separar os seletores usando uma virgula.

Assim não precisamos ficar copiando e colando os mesmos códigos para todos, nós podemos fazer uim só bloco de CSS e reutilizar as mesmas regras!
